
  # Portfólio Pessoal

  Este repositório contém o código-fonte do site de portfólio pessoal de Leo Filipe. O objetivo do site é apresentar o percurso académico e profissional, projetos, competências e contactos de forma simples, responsiva e otimizada para performance.

  O que encontrará neste repositório:

  - Seções principais: `Hero`, `Sobre`, `Experiências`, `Formações`, `Projetos`, `Contacto`.
  - Componentes React reutilizáveis e estilos com Tailwind CSS.
  - Pequenos utilitários e componentes de interface (botões, cards, modais, navegação).

  Tecnologias principais:

  - React 18
  - Vite (bundler e dev server)
  - Tailwind CSS
  - Motion para animações simples

  Como executar localmente

  1. Instalar dependências:

  ```bash
  npm install
  ```

  2. Executar em modo desenvolvimento (servidor local):

  ```bash
  npm run dev
  ```

  3. Gerar build de produção:

  ```bash
  npm run build
  ```

  4. Gerar e mover `dist` para `docs/` (usado para publicar no GitHub Pages):

  ```bash
  npm run build:publish
  ```

  Estrutura resumida do projeto

  - `src/` — código fonte React, componentes, estilos.
  - `imports/` — imagens e assets usados pelo site.
  - `scripts/` — scripts auxiliares (ex.: mover `dist` para `docs`).
  - `dist/` — build de produção gerado pelo Vite.
  - `docs/` — cópia do `dist` pronta para publicação no GitHub Pages (quando aplicável).
